## ✨ 前端介面優化

此版本針對現場掃描人員的使用者體驗進行了多項優化：

- **佈局調整**：將掃描結果置於畫面頂部，方便快速查看；將較少使用的 API Key 輸入框移至底部。
- **行動裝置優化**：禁止了在手機上點擊輸入框時的頁面自動縮放，避免相機畫面跑位。
- **即時視覺回饋**：在掃描成功或失敗時，整個頁面背景會閃爍綠色或紅色，提供強烈的視覺提示。
- **音效回饋**：為成功和失敗的掃描提供不同的音效，方便操作人員在不看螢幕的情況下也能得知結果。
